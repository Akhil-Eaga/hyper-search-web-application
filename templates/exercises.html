Search hashtags

Want to search for some popular trends? Try adding the "$hastag" in the search.

For example, if you want to find the hashtag "throwbackthursday"


#throwbackthursday

Technique 3:

Exclude words from search

Want to excludes some words in your search? Try adding the "- remove_this_word" in the search.

For example, if you want to search for mustang but not the car


mustang -car


Give it a try and notice the difference in the relevance of results.

Technique 4:

Search for exact match

Want to search for an exact phrase? Try wrapping the search in double quotes.

For example, if you want to find the Puppy Dog Sweaters, then use the following:


"Puppy Dog Sweaters"


If you don't wrap them, in quotes, the words can occur in any sequence in the search results

Notice the difference in the number of search results returned.

Technique 5:

Search on specific sites

Want to search for articles or content on specific site? Try adding the "site:website.com" at the end.

For example, if you want to search for Sidney Crosby on nhl.com, then use the following:


Sidney Crosby site:nhl.com


Notice the difference in the number of search results returned.

Technique 6:

Search for pages that link to specific page

Want to search for sites link to a specific site? Try using the "link:website.com".

For example, if you want to search for sites that link to Newyork times website, then:


link:nytimes.com


This search returns all pages that link to New York Times official website.

It is even hard to think of the search phrase for this criteria. Isn't it ?

Technique 7:

Search for similar sites

Want to search for sites similar to your favourite site? Try using the "related:website.com".

For example, if you want to search for sites that are similar to amazon.com


related:amazon.com


This search returns all pages are similar to amazon.com

Notice the difference in the number of search results returned.

Technique 8:

Search for multiple phrases

Want to search for multiple words or phrases at once? Try using the "OR".

For example, if you want to search for sites that contain "chocolate" or "white chocolate"


chocolate OR white chocolate


This search returns all pages that have either chocolate or white chocolate

Notice the increase in the number of search results returned.


Result of normal search Result of refined search

Technique 9:

Search for specific filetypes

Want to search for resources that are of specific filetype? Use the "filetype:type_of_file".

For example, if you want to search for pdf documents on programming resources


Programming Resources filetype:pdf


This search returns pages that have programming resources in pdf format

Notice the increase in the number of search results returned.

Technique 10:

Search using wildcard *

Recollect only a few words of a song but want to search for it? Use the "*".

For example, if you want to search for the song "Never Gonna Give You Up" but missed some words:


Never * Give * Up


This search returns resources by making the * match with any word.

Notice the increased relvance in results.
