Let's learn to search better on google.

  • Starting from this slide you will see a Google Search box with some important flags in the placeholder field for you to try out.
  • As you progress through each slide,type the flag into the search box to see how each flag affects your search result
  • For your convenience, the search box is prepopulated with the code but we encourage you to type your own personalized results to see it work!.
  • After typing in the search box click the "HyperSearch" button or simply press Return/Enter on your keyboard to load the search on a new tab.
  • Let's try searching for a smiling baby yoda's picture on google. Try typing baby-yoda-happy.
    Notice how "- hyphen " is used to link to each words.

Quickly find social media handle of someone.

  • Let's say you wanna follow Elon Musk on twitter to capitalize on the next Dogecoin wave but you don't remember this twitter handle.
  • Try adding the "@social-media-name" in the search along with the name query.
  • For example, try typing Elon Musk @Twitter.

Search information relevant to a #hashtag.

  • Try adding the "#" in the search along with your name query.
  • For example, if you want to find the hashtag #BillieEilish.

Exclude certain words from search result.

  • Let's say you want to find famous people named Steve other than Steve Jobs.
  • Given Steve Job's popularity, finding any one with a similar name will be challenging. One tip is to try removing Apple from search results.
  • Try adding the "- searchword" in the search. For example, Steve -Apple

Find results that match the exact keyword!

  • Want to search for an exact phrase? Try wrapping the search in double quotes.
  • For example, to find the belgian waffles recipes type "Belgian Waffles Recipes"

Find something on a specific site. Great way to search for hidden deals!

  • Want to search for articles or content on specific site? Try adding the "site:website.com" at the end.
  • Let's say you wish to buy the latest Virtual Reality (VR) Headset from amazon.com.au try, VR Headset site:amazon.com.au

Filter news by source.

  • Let's say, before investing in Google's stocks you want to find out whether the latest news about Google's upcoming technology is from a trusted source of yours or not.
  • One way of doing it is by using this "news-article source: source-name".
  • For example, google leaks source:9to5google.

Search for similar sites.

  • Bored using Facebook?
  • Let's search for websites similar to Facebook using the "related:website.com" flag.
  • For example, related:facebook.com

Search for multiple phrases

  • Let's say you want to search for images that are either "Dark Chocolate" or "Milk Chocolate"
  • Try Dark Chocolate OR Milk Chocolate.
  • You can also use AND to find results containing both together. (Note this one is sometimes a hit or miss)

Search for specific filetypes

  • Want to search for resources that are of specific filetype? Use the " filetype:type_of_file ".
  • For example, if you want to search for pdf documents on programming resources
  • Try searching for object oriented programming filetype: pdf

Search using wildcard *

  • Recollect only a few words from a song but want to search for it?
  • Use the ' * ' to search for the songs with missing lyrics
  • Type "never * give * up" and see what comes up!
